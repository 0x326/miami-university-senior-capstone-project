# scale-interface

## Usage

```bash
cd miami-university-senior-capstone-project/
cd packages/scale-interface/

# Building code
yarn build

# Running code
yarn start

# Testing code
yarn test

# Linting code
yarn lint

# Fixing lint errors
yarn lint --fix
```

## Technologies

- Runtime environment: [Node.js]
- Static type checking: [TypeScript]
- Unit test framework: [Jest]
